# Systems

This section provides clear, structured documentation for building systems and data centre infrastructure. It includes fundamentals from the Seneca Building Systems Engineering Technician program as well as real-world observations from data centre environments.

Each page focuses on practical understanding, component breakdowns, workflows, and engineering relevance.

## Included Systems

- HVAC — air handling, cooling, CRAC/CRAH units, chillers, pumps  
- Electrical — distribution, transformers, UPS, switchgear  
- Fire Protection — sprinklers, suppression, detection systems  
- Plumbing — water supply, drainage, pumps  
- Controls / BMS — monitoring, automation, sensors, control loops  
- Data Centre Infrastructure — white space, racks, cooling topology, power paths  
- Redundancy & Resilience — N, N+1, 2N, failover strategies  
- Safety Protocols — PPE, lockout/tagout, hazard identification
