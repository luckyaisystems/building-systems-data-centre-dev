# Building Systems + Data Centre Development

This repository documents my learning, coursework, and practical development across two domains:

- The Seneca College **Building Systems Engineering Technician** diploma program  
- Real-world exposure to **data centre operations**, workflows, and facility systems

The goal of this project is to create clear, accurate, and professional engineering documentation that reflects both academic training and practical observations. This includes system overviews, diagrams, SOPs, competency development, and internship notes.

## Repository Structure

### Systems
Documentation for building and data centre systems, including:
- HVAC
- Electrical
- Fire Protection
- Plumbing
- Controls / BMS
- Data Centre Infrastructure
- Redundancy and Resilience

### Labs and Coursework
Notes, assignments, and course mapping from the Seneca program.

### Internship
Observations, workflows, incident logs, and development notes from data centre environments.

### Diagrams
Mechanical, electrical, control, and topology diagrams used for understanding and documenting systems.

### SOPs
Structured procedures, DAAD cycles, competency mapping, and maintenance checklists.

## Purpose

This repository serves as a professional engineering portfolio and a long-term reference system. It is designed to demonstrate:
- technical understanding of building and data centre systems  
- disciplined documentation habits  
- structured workflows  
- real-world learning and development  
- readiness for facility operations and data centre environments
